# 💡 채팅 없는 즉시 거래 프로세스 정의

## 1단계: 판매자의 물품 등록 (조건 사전 정의)
* **행동**: 판매자가 '새 물품 올리기'를 진행합니다.
* **필수 입력**: 기존 상품 정보 외에 거래 가능 장소(예: 학생회관 1층, 7호관 로비 등)와 거래 가능 시간대(예: 오전, 점심, 오후)를 최소 1개 이상 체크박스 형태로 필수 선택하게 합니다.
* **상태**: 등록 완료 시 상품은 판매중 상태가 됩니다.

## 2단계: 구매자의 조건 선택 및 구매 요청 (채팅 대체)
* **행동**: 구매자가 상품 페이지에서 `[예약 신청하기]` 버튼을 누릅니다.
* **UI 변화**: 채팅창이 열리는 대신 하단 팝업(Bottom Sheet) 또는 모달 창이 뜹니다.
* **선택**: 팝업에는 판매자가 1단계에서 미리 설정해 둔 '장소'와 '시간대' 목록만 노출됩니다. 구매자는 이 중 자신이 가능한 장소 1곳과 시간대 1곳을 선택한 뒤 `[이 조건으로 거래 요청]`을 누릅니다.
